# To run
## React Native
- Navigate to patientSide directory
- Run either of the following commands
```
npm run android
npm run ios
npm run web
```

## SvelteKit
- Navigate to facilitySide directory
- Run ```npm run dev```

# In case of errors
- Run ```npm install``` inside the directory with the error then rerun command
